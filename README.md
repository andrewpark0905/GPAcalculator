# GPA Tracker

## Summary
A web app that calculates weighted and unweighted GPA from a list of classes (5.0 scale AP). Built by upgrading a high-school project of JavaScript GPA calculator app into a full web application with a backend and database.

## Features included
- GPA calculator (weighted + unweighted)
- Dynamic add/remove class rows
- Responsive UI (mobile + desktop)
- Deployed as a web app (Express serves the frontend)

## Tech stack
- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js + Express (static hosting)

## Time spent
3-4 hours

## Run locally
1. npm install
2. npm start
3. Open: http://localhost:3000
